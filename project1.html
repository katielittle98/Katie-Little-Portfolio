My Movie Browser

This page contains the case study for my movie browser website, which includes a project overview, video demonstration, tools used, and links to the project.

Video Demonstration

Project Overview

For this project, I created a movie browser website that leverages the power of a search engine integrated with the OMDB (Open Movie Database) API, a RESTful web service, which extracts movie information such as movie titles, release dates, plot summaries, ratings, and more. This website allows users to explore a vast collection of movies and easily find information about their favorite films. Additionally, users have the option to create a personal watchlist, where they can add movies of interest and save them to watch later.
