SOAP MAT Website

This page contains the case study for the website I created, SOAP MAT, which includes a project overview, tools used, and links to the project.

Project Overview

SOAP MAT, which stands for Stop Opioid Addiction Program, Medication-Assisted Treatment, is based in San Diego, CA. Their facility offers medication-assisted treatment, counseling, case-management, youth services, and perinatal services.
