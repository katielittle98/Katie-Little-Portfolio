Get to know me!

My name is Katie Little and I live in San Diego, CA. I'm a University of Oregon Alumnus (Go Ducks!). I've been learning how to code and build web applications since 2020. My interest in coding first began in 2013 when I was an active Tumblr user. I thoroughly enjoyed editing the HTML associated with my profile to make my page appear both interactive and exciting. Since then, I have advanced my knowledge in HTML, CSS and Javascript in order to build responsive web designs.

I find joy in the process of problem-solving and seeing my code come to life. My commitment to continuous learning and dedication to mastering coding skills have led me down a rewarding path towards a career in the technology industry.
